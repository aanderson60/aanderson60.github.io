Group Testing Research

Group testing is a strategy to test a set of samples in a way that eliminates the need to test each individual within the set. This is done by pooling groups of samples and testing those groups in an organized, algorithmic way. The strategy intends to minimize false negatives while reducing the number of tests done on the set. There are two generalized methods for group testing schemes (Aldridge et al., 2019):

  • Adaptive Testing
    • Multiple stages that depend on each other
    • Stages must be run sequentially
  • Non-adaptive testing
    • One or multiple independent stages
    • Stages can be run in parallel
    • Usually much faster

This research project intends to explore the application of group testing theory in the context of food safety working with salmonella in a set of samples. More specifically, the primary objectives are:

  • To show that non-adaptive group testing schemes can be used to identify salmonella in a sparsely infected population (a prevalence of 1-3%).
  • To expand upon the current theories of group testing in the framework of food science and understand the limitations of the theory when working with diluted samples of salmonella.
  • To explore new ideas and strategies to use group testing to reduce time and money demands of testing populations for bacteria in food science.
